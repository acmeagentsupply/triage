TRIAGE REPOSITORY DOCTRINE

1. This repository has a single defined purpose: providing a safe, read-only diagnostic tool for OpenClaw operators.
2. This is an open-source community tool. No internal infrastructure, private repo names, or corporate system references belong here.
3. All code must honour the Trust Doctrine: read-only, no telemetry, no network calls, proof-bundle only.
4. Do not add collectors that call paid or proprietary services — those belong in the commercial edition.
5. If a file's purpose is unclear, stop and consult the README before editing.
6. Never duplicate operational code from private infrastructure into this repository.
